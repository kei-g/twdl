# Twitter Images Downloader [![license][license-image]][license-url]

[`twdl`][github-url] - Twitterの画像をダウンロードするアプリケーションです.

## Supported Platforms _動作環境_

下記のOSで動作します

- Windows
- Linux

:warning: Executable only on 64bit architectures _64bit環境でのみ動作します_
